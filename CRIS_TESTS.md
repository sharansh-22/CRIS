## TS-001: Entropy Method Selection

### Why This Test Was Conducted
To select the best entropy method for detecting market regime changes before black swan events.

### Tests Conducted
Shannon, Permutation, Sample, Tsallis entropy methods tested across COVID-2020, Flash Crash 2010, China 2015.

### Candidates Evaluated
| Candidate | Method Name |
|-----------|-------------|
| 1 | Shannon Entropy |
| 2 | Permutation Entropy |
| 3 | Sample Entropy |
| 4 | Tsallis Entropy |

### Evaluation Metrics
- Lead Time
- False Positive Rate
- Signal-to-Noise Ratio
- Directional Consistency
- Stress Duration
